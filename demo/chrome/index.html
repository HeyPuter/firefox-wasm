Firefox in WebAssembly

This is the gecko engine compiled to WebAssembly, running the actual firefox UI in a web browser. All js execution and rendering is done inside webassembly, instead of the host browser.

Assets Downloaded once per visit, served from memory.
Engine Firefox/Gecko compiled to WebAssembly.
Network Tabs load web content through a Wisp server.
